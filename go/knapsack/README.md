## 部分背包
给定 n 种物品和一个容量为 C 的背包，物品 i 的重量是 wi，其价值为 vi，每个物品都有ki件，现在往背包里面装东西，怎么装能使背包的内物品价值最大？
一个比较简单但是效率不高的方法： 把物品i的ki件物品转化为k件不同的物品，直接就转化为01背包问题了。不过效率较低。
另一种思路：再申请一个数组来保存物品i的使用个数，从而控制物品i的数量,用完全背包的思路解决问题
